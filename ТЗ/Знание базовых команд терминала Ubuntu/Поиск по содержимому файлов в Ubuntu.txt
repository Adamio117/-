Базовая команда
grep "искомый_текст" /путь/к/файлу


Рекурсивный поиск по всем файлам в директории и ее поддиректориях
grep -r "ERROR" /var/log/
Ищет слово "ERROR" во всех файлах внутри /var/log и его поддиректорий

Поиск без учета регистра
grep -i "mysql" /etc/config.txt
Найдет "mysql", "MySQL", "MYSQL" и т.д.

Показ номера строки, в которой найдено совпадение
grep -n "password" config.yml
Выведет matched строку и ее номер в файле

Поиск с использованием регулярных выражений
grep -E "^[0-9]{3}-[0-9]{2}-[0-9]{4}" data.txt
Ищет строки, начинающиеся с шаблона типа SSN (123-45-6789)

Поиск во всех файлах с определенным расширением
grep -r --include="*.php" "function calculate" /home/user/project/
Ищет "function calculate" только в PHP-файлах внутри проекта

Инвертированный поиск
grep -v "#" script.sh
Покажет все строки файла script.sh, которые не являются комментариями
